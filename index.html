常见问题

PowerShell 提示"无法识别"或"安全错误"？
先运行 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser，输入 Y 确认，再重新粘贴安装命令。
装完了但终端输入 claude 找不到？
把 C:\Users\你的用户名\.local\bin\ 加到系统 PATH。具体：开始菜单搜「环境变量」→ 编辑系统环境变量 → 环境变量 → Path → 新建 → 粘贴路径 → 确定，重开终端。
不装 Git Bash 可以吗？
不可以。Claude Code 在 Windows 上依赖 Git Bash 运行。按上面第 1 步下载安装，有 64-bit 和 ARM64 两个版本，选对应你电脑的即可。不确定就选 64-bit。
下载特别慢或者卡住了？
下载约 200MB+，取决于网络。反复失败请检查能否访问 storage.googleapis.com。
如何卸载？

设置 → 应用 → 已安装的应用，搜索「Claude Code」卸载。

完全清除配置：

powershell Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force Remove-Item -Path "$env:USERPROFILE\.claude" -Recurse -Force Remove-Item -Path "$env:USERPROFILE\.claude.json" -Force
提示 command not found？
运行 source ~/.zshrc（zsh）或 source ~/.bashrc（bash），或者关掉终端重新打开。Mac/Linux 安装脚本一般会自动配置 PATH。
提示权限不足？
Mac/Linux 一般不需要 sudo。如果遇到权限问题，检查 ~/.local/bin/ 目录是否存在且可写。
下载特别慢或者卡住了？
下载约 200MB+，取决于网络。反复失败请检查能否访问 storage.googleapis.com。
如何卸载？

终端运行：

bash rm -rf ~/.local/bin/claude rm -rf ~/.local/share/claude rm -rf ~/.claude rm -f ~/.claude.json
这脚本安全吗？
在浏览器直接打开脚本链接查看源码。脚本使用国内 npm 镜像（npmmirror.com）下载，不经过第三方中转，不收集数据。
